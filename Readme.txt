# Basic-Banking-System

Project developed during my Internship at The Sparks Foundation.

# AIM
Create a simple dynamic website which has the following specs.

Start with creating a dummy data in database for upto 10 users. Database options: Mysql, Mongo, Postgres, etc. 

Flow: Home page-> user_list-> Transfer money-> Show Transaction history.

No Login Page. No User Creation. Only transfer of money between multiple users.

# Tech Stack
FRONT END: HTML, CSS, JavaScript.

BACK END: PHP

DATABASE: MySQL
